“合计啥呢，以后你也得埋在这”

这句话很有趣，不但没有让我感到死亡的恐惧，反倒有一种知道自己结局的踏实。

那之前，在一件事上，我经常会想“晚不晚、“敢不敢”这样的娘炮问题。

那之后，“只要我想”、“这事应该”。满足这两点，那就毫不犹豫的干，就像真人快打里那样“拳拳到肉”的干。

对我来说，自己埋哪都知道了，买比特币这种有趣的事，有什么好犹豫的。

不买比特币增值，难道去上班么？
